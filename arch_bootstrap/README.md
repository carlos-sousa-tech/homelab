# Arch Linux Bootstraper

## Using the Auto Installer
- Get the auto-installer to your system
    - scp: `pacman -Sy openssh` > `passwd` > `scp install_arch.sh root@ip:~/`
    - curl: `curl -O URL` > `chmod +x install_arch.sh` > `./install.sh`
